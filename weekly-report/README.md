# 周报助手使用说明

## 功能简介

周报助手自动记录每日工作内容，在用户要求时按四大模块生成结构化周报。

## 四大模块

- **新工具**：本周新引入的工具、系统、机制
- **主要推动**：核心工作事项、问题解决、效果反馈
- **下周计划**：下一阶段规划
- **AI应用**：AI/智能工具相关进展

## 使用方式

### 1. 记录工作

当输入以「今日总结：」开头时，自动记录当天工作内容。

示例：
```
今日总结：测试在线客服工具，修复3个错误，准确率提升至92%
```

回复：`✅ 已记录今日工作（周X）`

### 2. 生成周报

当输入「生成周报」「本周总结」或「周报」时，自动生成本周工作总结。

示例：
```
生成周报
```

## 关键词自动归类

系统会根据关键词自动将记录归类到对应模块：

| 模块 | 关键词 |
|------|--------|
| 新工具 | 新工具、上线、引入、平台、系统、机制、部署、搭建 |
| 主要推动 | 推动、组织、解决、优化、测试、修复、培训、制度、流程、SLA、同步、完成、沟通、会议、需求、开发 |
| 下周计划 | 计划、准备、规划、下一步、推进、继续、跟进、即将、下周、后续、安排 |
| AI应用 | AI、智能、模型、语料、助手、客服工具、小锁、准确率、训练、大模型、LLM |
