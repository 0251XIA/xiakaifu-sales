# Slase 销售能力体系

> 公有云销售，5年经验，服务企业超过2000家

## Overview

Slase 销售方法论是一套完整的电子签章销售体系，核心是**销售闭环七阶段**配合**个人四步法**，通过结构化的方式完成从线索到续费的全流程。特点是真诚沟通、以顾问姿态服务客户。

## When to Use

**Use when:**
- 销售推进遇到瓶颈，不知道下一步怎么走
- 客户在价格谈判中反复压价
- 演示不知道讲什么、怎么讲
- 客户犹豫不决，不知道怎么逼单
- 续费时客户说要考虑
- 不知道如何挖掘客户需求
- 客户一开口就说要报价
- 客户说友商更便宜

**Don't use when:**
- 客户需求完全不匹配产品能力（应转线下或放弃）
- 客户已经有明确竞品在做深度对比（需要更专业的方案团队）

## 产品学习四步法

1. **定位功能** — 清晰每个功能点位置
2. **理解作用** — 知道能解决什么问题
3. **掌握逻辑** — 理解背后业务逻辑
4. **串联功能** — 由点到线到面，构建体系
5. **升华** — 想象产品如何改进，提建议

## 核心谈判技巧

| 技巧 | 应用场景 | 做法 |
|------|----------|------|
| 甩锅法 | 价格太低 | "价格比较低，商务他不批" |
| 降价不过两 | 不能一直让价 | 第一次正常沟通，第二次领导特批，没有第三次 |
| 赠券用起来 | 单价不降，赠券变相降价 | "同样价格可以多用XX份" |
| 三档报价 | 主推第三档 | "红花需要绿叶配"，二三档差距要大 |
| 以退为进 | 站在客户角度给建议 | "低价无利润，服务跟不上" |
| 预设埋点 | 不可一降再降 | 单价是心理防线 |

## 演示三段论

**"挖坑→埋土→种花"**

1. **挖坑**：结合客户场景和痛点，吸引注意
2. **埋土**：告知如何解决，让客户放心
3. **种花**：价值升华，带来额外惊喜

## 复杂案例处理

**客户用友商、对我们评价不高时**：
1. 长期跟进，定期发新功能，保持沟通，寻找机会
2. 以顾问形式沟通，懂友商不比友商销售差
3. 赠送测试名额，让客户重新测试，用专业性扭转判断
4. 按行业最佳实践配置，体现便利性
5. 部分业务先免费试用，逐步切过来
6. 全程不推销，处处替客户考虑

## 常见问题处理

| 问题 | 处理方法 |
|------|----------|
| 客户一开口就说要报价 | 不直接给，先搞清楚客户真实需求。问他需要你做什么 |
| 客户说友商更便宜 | 传递价值：便宜不代表好用，价值决定价格 |
| 客户采购卡住 | 没有好方法，保持真诚沟通，增加交流演示机会 |
| 业绩不好时 | 鼓励自己，认真联系每个客户，学习提升专业性 |

## 常见错误

**问题：** 听不懂客户需求就推荐产品
**修复：** 先用客户画像五要素搞清楚真实需求

**问题：** 只会报价不会讲价值
**修复：** 价值决定价格，先传递价值再谈报价

**问题：** 客户说"选你们了，给个报价"直接给报价
**修复：** 保持警惕，客户可能是在比价，先搞清楚需求

**问题：** 只引导下单不解决客户需求
**修复：** 以顾问姿态服务，处处替客户考虑
